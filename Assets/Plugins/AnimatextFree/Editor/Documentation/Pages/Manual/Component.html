Component

Create

Go to "Inspector" panel, click "Add Component" and select "Scripts - Animatext Free", then select a version and start to create.

Header

  • New EffectClick it to add a new effect in the first place.

  • DocumentationClick it to view the dcumentation.

  • Settings Click it to enter or exit system settings.

Settings

  • Retained TextThe retained text content in the text component after Animatext parsing.

    • Input TextThe text initially entered of the text component.

    • Parsed TextThe parsed text of animatext.

    • Output TextThe parsed text of the text component.

    • Effect TextThe text containing the effect.

    • Blank TextNo text included.

    For example:<effect><b>Anima<b></effect><c>text</c>, <effect> is effect tag, <c> is unit tag, <b> is text component tag.

    • Input Text<effect><b>Anima<b></effect><c>text</c>

    • Parsed Text<b>Anima</b>text

    • Output TextAnimatext

    • Effect TextAnima

    • Blank Text

    Tip: It is recommended to select blank text as it helps reduce the performance cost of generating original vertices.

    Note that: When the retained text is undone in editor playing mode, the text for effect execution is the text in the component prior to the undoing instead of the text input prior to the undoing.

  • Tag Symbols

    • Angle BracketsUsing angle brackets "<" and ">" to denote tag symbols.

    • Round BracketsUsing round brackets "(" and ")" to denote tag symbols.

    • Square BracketsUsing square brackets "[" and "]" to denote tag symbols.

    • Curly BracketsUsing curly brackets "{" and "}" to denote tag symbols.

  • Marker Symbols

    • SlashesUsing the slashes "/" to denote marker symbols.

    • BackslashesUsing the backslashes "\" to denote marker symbols.

    • VerticalBarsUsing the vertical bars "|" to denote marker symbols.
