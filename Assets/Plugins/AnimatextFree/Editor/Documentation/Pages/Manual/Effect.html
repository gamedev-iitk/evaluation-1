Effect

Parameter

  • TagThe tag name of the effect. Each name best be made of 2 or more letters or numbers. Note that:

    • The tag name can't contain tag symbols or marker symbols, by default including "<", ">", "/".

    • The tag name can't be the same as the name of a unit tag, including "c", "w", "l", "g".

    • The tag name shouldn't be the same as the rich text tag name of the component.

      UGUI - b, color, i, material, quad, size

      If it is used when rich text is enabled within the component, then the rich text effect will not be applied on the text, but the animation effect will not be affected.

  • Auto StartThe effect enters the start status automatically.

  • Auto PlayThe effect enters the play status automatically.

  • Refresh modeRefresh mode determines the state of the effect when the component is refreshed.

    • StartThe effect restarts after refreshing a component.

    • PlayThe effect replays after refreshing a component.

    • PauseThe effect pauses after refreshing a component.

    • ContinueThe effect continues after refreshing a component.

    • EndThe effect ends after refreshing a component.

Execution

State

  • StopThe effect is not executed and is ready to start.

  • StartThe effect is executed at the time of 0.

  • PlayThe effect is executed at the effect time and is played.

  • PauseThe effect is executed at the effect time and is paused.

  • EndThe effect is executed at the time of positive infinity.

Order

Multiple effects of the same text will be executed in the order of the effects from first to last.
