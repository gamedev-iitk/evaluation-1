Installation

Open your project. Go to "Project" panel, right-click and select "Import Package - Custom Package", then select Animatext Free Package.

How To Use

  • 1 - Create presetGo to "Project" panel, right-click and select "Create - Animatext Free Preset", then click on any item to create.

  • 2 - Add componentGo to "Inspector" panel, click "Add Component" button, and select "Scripts - AnimatextFree - Animatext Free UGUI".

  • 3 - Add effectClick "New Effect" button in Animatext Free.

  • 4 - Set tagSet tag name in the effect

  • 5 - Set presetSet preset in the effect

  • 6 - Add tagsAdd tags in the text component
