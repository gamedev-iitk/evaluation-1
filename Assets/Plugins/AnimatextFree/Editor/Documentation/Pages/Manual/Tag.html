Tag

Basic

Tags in animatext are similar to those in rich text. If you are not familiar with rich text, please read UGUI - Rich Text.

  • Opening tagMarks the opening of a section, <tagName> by default.

  • Closing tagMarks the closing of a section, </tagName> by default. The closing tag at the end of the text can be omitted.

  • Empty tagMarks empty content of a section, <tagName/> by default. The abbreviation of <tagName></tagName>.

  • Single closing tagMarks the closing of the previous opening tag, </>by default.

  • All closing tagMarks the closing of all opening tags before it, <//>by default.

Unit Tag

Each unit tag defines the text it encloses.

  • <c>It defines the enclosed text as a new character in character presets.

  • <w>It defines the enclosed text as a new word in word presets.

  • <l>It defines the enclosed text as a new line in line presets.

  • <g>It defines the enclosed text as a new group in group presets.

Added unit tag splits up the original unit.

For example: Ani<w>ma</w>text would mean the "ma" is defined as a whole word. As such, upon execution, the original word "Animatext" would be split into three separate words: "Ani", "ma", "text".

Unit tags do not support nesting.

For example: <w>Ani<w>ma</w>text</w> is the same as <w>Animatext</w>.

Effect Tag

Effect tags are used to customize the effects. You can set the name of the tag within the effect of the component.

Each name best be made of 2 or more letters or numbers. Note that:

  • The tag name can't contain tag symbols or marker symbols, by default including "<", ">", "/".

  • The tag name can't be the same as the name of a unit tag, including "c", "w", "l", "g".

  • The tag name shouldn't be the same as the rich text tag name of the component.

    UGUI - a, b, color, i, material, quad, size

Notation

By default, Animatext uses angle brackets("<" , ">") to denote tag symbols and slashes ("/") to denote marker symbols. You can change the default notations for a component.

For example: Using curly brackets ("{" , "}") to denote tag symbols and backslashes ("\") to denote marker symbols.

  • Opening tag{tagName}

  • Closing tag{\tagName}

  • Empty tag{tagName\}

  • Single closing tag{\}

  • All closing tag{\\}
