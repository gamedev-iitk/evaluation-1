Third Party

Precaution

Pay attention to the following before using any third-party plug-ins:

  • 1 - Turn off the built-in text effects in third-party plug-ins.

    Animatext uses an independent effect computation method, which is not compatible with the text effects of other third-party plug-ins.

    Before using Animatext, turn off the text effects of third-party plug-ins.

  • 2 - Turn off the function of modifying the text in a third-party plug-in.

    The text effects in Animatext are to be analyzed based on the text in the component. Modifying the text in the component may cause the text effect to be abnormal.

    Prior to operation of Animatext, turn off the function of modifying the text in a third-party plug-in.

    For example: the commonly used typewriter function in the dialog plug-in will modify the text in the component.

    To use this type of plug-in, turn off the typewriter function, or display all the text at once.

  • 3 - Avoid tag conflicts.

    If there is any conflict between Animatext default tags and third-party plug-in functions, the tag symbols and marker symbols in Settings may be modified.

More details about how to use a third-party plug-in will be updated gradually. If you have any questions about the use of third-party plug-ins, please contact us!
